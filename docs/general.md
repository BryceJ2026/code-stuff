## Just some general tips for coding

* The three kinds of variables in java and python are integers(numbers), Strings(words), and boolean(true or false)
* Use GitHub if you want the best results while coding
* If something does not work, look it up and figure it out
* If you use code that is not yours, credit them in the file
* Using AI to code is okay, but it is better to use it to do tedious tasks, like compiling information
* Make sure you understand what the code is doing, so you can use that info later
* Be very creative, and always think outside the box.
